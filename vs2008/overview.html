leptonlib is a library of image processing routines written in C. This document discusses linking with pre-built Windows 32-bit binaries to create working executables with Microsoft’s Visual Studio 2008. A brief section on using Visual Studio 2010 is also included.

leptonlib can process a number of image file types. The supplied static and dynamic libraries support the JPEG, PNG, TIFF, and GIF file formats.

Also available is a Visual Studio 2008 solution that lets you build the library and the executables in the Leptonica /prog directory. It contains a complete hierarchy of all the /prog directory programs listed by type, category, and filename to make browsing the many files easier.

Detailed instructions are given on rebuilding the supplied image libraries.
